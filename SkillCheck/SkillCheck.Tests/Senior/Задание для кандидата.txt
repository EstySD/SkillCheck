Ознакомление:
- В папке TestInfrastructure находятся классы для запуска веб-сервера. 
  Эти классы понадобятся для работы с интеграционными тестами.
- В папке IntegrationTests расположен класс ExampleTest, 
  который демонстрирует пример интеграционного теста для обращения к API экземпляра приложения.
- В той же папке находится класс NoteTests, в котором вам необходимо описать все необходимые тесты.

Задание:
- В классе NoteTests вам необходимо написать тесты, 
  которые проверяют выполнение следующих бизнес-условий. Тесты должны быть интеграционными, 
  обращающимися к API, а также юнит-тестами при необходимости.

Бизнес-условия:
- Создание и удаление блокнота. Блокнот можно создать на основе шаблона.
- Получение всех блокнотов
- Просмотр содержимое блокнота. У блокнота должен быть заголовок, который отображается в списке блокнтов. 
  Содержимое блокнота может состоять как из обычного текста, так и из элементов типа checkbox (как задачи).
  Реализовать возможность задания текста со следующими стилями: жирность текста, цвет текста.
  Текст должны поддерживать перемещение вверх и вниз по странице, как отдельные элементы.
- Создание шаблонного блокнота.
- Удаление шаблонного блокнота.
- Редактирование шаблонного блокнота.
- Сортировка блокнотов по размеру блокнота.

Дополнительные указания:
- Реализацию бизнес-логики вы можете выполнить в проекте SkillCheck, 
  либо по вашему усмотрению, если хотите разделить проект на отдельные модули. 
  Это решение также будет учитываться при оценке вашего задания.
