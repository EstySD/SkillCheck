Ознакомление:
- В папке Intern есть две папки с двумя уровнями:
  EasyLevel (required) - простой уровень для обязательного выполнения.
  HardLevel (required) - сложный уровень, тоже, для обязательного выполнения.

Задание:
- В папке EasyLevel (required) необходимо раскомментировать тесты и описать реализацию, чтобы эти тесты выполнялись. 
  То есть необходимо самим создать классы, методы и аргументы к этим методам.
- В папке HardLevel (required) тесты уже описаны, но необходимо описать только реализацию


Дополнительные указания:
- Реализацию бизнес-логики вы можете выполнить в проекте SkillCheck, 
  либо по вашему усмотрению, если хотите разделить проект на отдельные модули. 
  Это решение также будет учитываться при оценке вашего задания.
